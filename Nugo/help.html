Logo Commands

Forward (fd) - Move the turtle forward in the direction it is pointing.
Back (bk) - Move the turtle backward.
Right (rt) - Turn the turtle clockwise.
Left (lt) - Turn the turtle counter-clockwise.
Repeat - Repeat commands a given number of times.

How to use

To tell the turtle to perform a command, click on its icon (on the right).

You can also type in commands in the command-line below the canvas.
E.g.: fd 100 <enter>

You can view and edit the code of the current drawing by clicking the Display Code button.
